# 形式語言與自動機 (Formal Languages and Automata)

- 課程號: 40240432
- 學分: 2
- 學時: 32

## 課程簡介

有限狀態自動機, 正規表達式, 正規語言; 上下文無關文法, 下推自動機, 上下文無關語言; 圖靈機, 計算理論初步知識

## 進度安排

- 課程概況及預備知識 - **2 學時**
- 有限狀態自動機, 正規語言, 正規表達式 - **12 學時**
- 上下文無關文法, 上下文無關語言, 下推自動機 - **12 學時**
- 圖靈機與計算理論初步 - **6 學時**

遇公休日或考試佔用等, 進度將做適當調整

## 考核方式

課後練習隨堂佈置, 以課本中的練習為主, 配合以適當的思考題

期中考試一次, 通常在第九周進行

### 成績比例

- 平時成績 **10** 分 (作業完成與平時表現)
- 平時小測 **20** 分
- 期末考試 **70** 分
